DESIGN SYSTEMS UTILS

Design Systems Utils aims to provide a series of common utilities at the beginning of any web project, which will facilitate and speed up the day-to-day development. For more information click on the following link.

Configuration

Initially it will be necessary to install the @front-tech/design-systems-utils dependency by executing the following command npm install @front-tech/design-systems-utils

For the correct functioning of the library it is necessary to create a configuration file in the project called .frontech.json, which will determine the utilities of margin, padding and custom grid. This configuration must maintain the following structure:

To use the library, it will simply be necessary to create an npm script with the following syntax:

design-systems-utils .frontech.json

And then import the following file to your sass stylesheet:

@use '~@front-tech/design-systems-utils/library/web/abstracts' with ($font-path:'public/assets/fonts/');

We can also generate a css output of the library specifying in the configuration file the path where we want to export said file:

"outputCSS":{
"path": "style.css"
}

fonts

font-face (mixin) 


                                
                                    $mixin
                                    font-face(
                                    $name, $path, $weight, $style, exts)
                                
                                
                                    
	$src: null;

	$extmods: (
		eot: "?",
		svg: "#" + tools.str-replace($name, " ", "_")
	);

	$formats: (
		otf: "opentype",
		ttf: "truetype"
	);

	@each $ext in $exts {
		$extmod: if(map.has-key($extmods, $ext), $ext + map.get($extmods, $ext), $ext);
		$format: if(map.has-key($formats, $ext), map.get($formats, $ext), $ext);
		$src: append($src, url(string.quote(settings.$font-path + $name + "." + $extmod)) format(string.quote($format)), comma);
	}

	@font-face {
		font-family: string.quote($name);
		font-style: $style;
		font-weight: $weight;
		src: $src;
	}

Descripción

Mixin whose objective is to create a source starting from an established route

Parametros

Nombre Descripción Tipo Valor por defecto
$name Font name. string
$path Font path. string
$weight Font weight. string
$style Font style. string
$exts Font format. string eot woff2 woff ttf svg

font-path (variable)

                                     'assets/fonts/'

Descripción

Variable path by default of the sources defined in the .frontech.json file. To modify the path, simply set the variable in the import as follows: @use 'frontech-library/library/web/abstracts' with($font-path:'public/assets/fonts/');

get-grid (mixin) 

                                
                                    $mixin
                                    get-grid(
                                    breakpoints)
                                
                                
                                    
  @each $break, $breakpoint in $breakpoints {
    $columns: map.get($breakpoint, "columns");
    @media only screen and (min-width: map.get($breakpoint,"width")) {
      @for $i from 1 through $columns {
        $valor: unquote((100 / $columns * $i) + "%");
        .col-#{$break}-#{$i} {
          box-sizing: border-box;
          padding: 0 map.get($breakpoint, "gutter");
          flex: 0 0 $valor;
          min-width: $valor;
          max-width: $valor;
        }
      }
    }
  }

Descripción

Mixin that, starting from the breakpoints established in the configuration file, generates the classes of the grid system.

    get-utilities-spacing (mixin) 

                                    
                                    @mixin
                                    get-utilities-spacing(
                                    $breakpoints, spacing)
                                
                                
                                    
  @each $break, $breakpoint in $breakpoints {
    @media only screen and (min-width: map.get($breakpoint,"width")) {
      $limit: map.get($spacing, "limit");
      @each $type, $property in $types {
        @for $i from 1 through $limit {
          $value: string.unquote($i*map.get($spacing,increase) + "px");
          .#{$type}-#{$break}-#{$i} {
            #{$property}: $value;
          }
        }
      }
    }
  }

    Descripción

    Mixin that, starting from the breakpoints established in the configuration file, generates the classes of the grid system.

    Parametros

    $spacing Map set in initial setup map $spacing

    str-replace (function) 

                                    
                                    
                                
                                
                                    
$index: str-index($string, $search);

@if $index {
	@return string.slice($string, 1, $index - 1) + $replace + string.replace(string.slice($string, $index + str-length($search)), $search, $replace);
}

@return $string;
